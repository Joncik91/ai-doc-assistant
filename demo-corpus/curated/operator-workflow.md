# Operator workflow

## Sign in

Operators can authenticate with either:

- JWT login using the seeded admin credentials
- `X-API-Key` for automation or scripted use

## Manage sources

- Upload TXT, Markdown, PDF, or DOCX files
- Review duplicate detection and ingestion warnings
- Delete documents when they are no longer needed

## Ask questions

- Use the chat panel to submit a grounded question
- Guardrails check the prompt before retrieval
- Answers include citations and a confidence indicator
